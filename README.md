# NFL Extension compatible with TurboWarp
A fully-compatible extension for TurboWarp with past, present, and future NFL games.
## How to install:
- Press the green '<> Code' button
- Press 'Download ZIP'
- Go to https://desktop.turbowarp.org/ and press download
- Open the TurboWarp file and it should come up as a new window
- Press File -> Import from computer -> 'NFL Live Tracker.sb3' (MAKE SURE YOU CHOOSE THE NEWEST VERSION)

Now you are done! Press the green flag and start tracking games!
